# E-Invitation / E-Greeting Card 

This project is a simple and interactive **web-based e-invitation / e-greeting card**
created for welcoming freshers. It was designed as a fun frontend project and
hosted online.

## Overview
The web page displays a visually appealing greeting card with interactive
elements. It can be shared using a link, making it suitable for digital
invitations and greetings.

## Tech Stack
- HTML

## Features
- Clean and responsive design
- Interactive greeting elements
- Lightweight and fast loading
- Easily shareable via link

## Hosting
This project is hosted using **GitHub Pages**.

## Use Case
- Fresher welcome invitations
- Digital greeting cards
- Frontend practice project

## Academic / Personal Context
Created as a personal web project during college to practice frontend
development and deployment.
